# React + Next.js 学习路径文档

欢迎来到 React + Next.js 学习路径！这是一个完整的、由简入深的学习方案，帮助你掌握现代前端开发。

## 📚 学习路径概览

本课程共分为 5 个阶段，涵盖 27 个核心概念：

### Phase 1: React Hooks 基础 ⭐
学习 React 19 的核心 Hook，为现代 React 开发打下坚实基础。

**涵盖内容：**
- `useState` - 基础状态管理
- `useEffect` - 副作用处理
- `useContext` - 全局状态管理
- `useReducer` - 复杂状态逻辑
- `useMemo` - 性能优化
- `useCallback` - 函数缓存
- `Custom Hooks` - 自定义Hook

**学习时间：** 3-5天

---

### Phase 2: Next.js App Router 🚀
掌握 Next.js 15 的全新 App Router 架构。

**涵盖内容：**
- 基础路由与文件即路由概念
- Layouts 布局系统
- 动态路由 [id] 参数
- 数据获取 Server/Client组件
- Navigation 导航系统
- Metadata 元数据管理

**学习时间：** 4-6天

---

### Phase 3: React 进阶技术 📚
深入理解 React 高级特性和复杂模式。

**涵盖内容：**
- `useRef` - DOM引用和可变值
- `useImperativeHandle` - 暴露方法给父组件
- `useLayoutEffect` - 同步副作用
- 复杂状态管理模式

**学习时间：** 3-4天

---

### Phase 4: Next.js 进阶应用 ⚡
学习 Next.js 的高级功能，构建生产级应用。

**涵盖内容：**
- API Routes - 后端接口开发
- Server Actions - 服务器动作
- Middleware - 中间件
- Caching - 缓存策略
- Streaming - 流式渲染

**学习时间：** 5-7天

---

### Phase 5: Tailwind CSS 🎨
学习 Tailwind CSS 4，快速构建现代化 UI。

**涵盖内容：**
- 工具类基础用法
- 响应式设计
- 深色模式实现
- 自定义配置
- 常用布局模式

**学习时间：** 2-3天

---

## 🎯 学习方法

### "先演示后实现"
每个概念都包含：
1. **交互式演示** - 直观了解功能效果
2. **代码展示** - 带中文注释的完整代码
3. **详细说明** - 使用场景、最佳实践、常见陷阱

### 循序渐进
- 每个阶段建立在前一个阶段的基础上
- 从简单概念开始，逐步深入复杂应用
- 提供充足的练习和示例

## 💡 学习建议

1. **按顺序学习** - 建议按照阶段的顺序进行学习
2. **动手实践** - 每个示例都要自己打一遍代码
3. **阅读注释** - 仔细阅读代码中的中文注释
4. **查阅文档** - 遇到问题时查看对应的文档说明
5. **完成练习** - 每个阶段后完成相关的练习
